# SM Dashboard Prototype

교통공사(SM) 운영 대시보드 제안용 프로토타입. React/Vite/Tailwind 기반 SPA로 KAC 프로젝트와 동일한 스택으로 구성했습니다.

## Quick Start

```bash
npm install
npm run dev
```

## Project Map

- App entry: `src/App.jsx`
- Global CSS, Pretendard: `src/styles.css`
- Vite config: `vite.config.js`
- Tailwind tokens: `tailwind.config.js`
